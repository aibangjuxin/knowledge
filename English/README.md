# English 知识库

## 目录描述
本目录包含英语学习、技术文档写作、邮件沟通和表达技巧相关的知识。

## 目录结构
```
English/
├── docs/                     # Markdown文档
├── godfather/                # 《教父》相关内容（可能用于学习）
├── scripts/                  # Shell脚本
└── README.md                 # 本说明文件
```
