# OpenAI & AI Tools 知识库

## 目录描述
本目录包含OpenAI及其相关AI工具、模型、平台的使用经验和知识。

## 目录结构
```
OpenAI/
├── docs/                     # Markdown文档
├── html/                     # HTML文档
├── images/                   # 图片资源
├── lists/                    # AI工具列表文件
└── README.md                 # 本说明文件
```

## 子目录说明 (按功能分类)
### docs/ - 文档
- `chattts.md`: ChatTTS语音合成相关
- `concept.md`: AI概念相关
- `English.md`: AI相关的英语内容
- `trans.md`: 翻译相关
- `Verifying-GLB.md`: GLB验证说明
- `web-development.md`: Web开发相关
- `pic.md`: 图片资源说明
- `Share.md`: 分享相关

### images/ - 图片资源
- `aistudio.png`: AI Studio相关图片
- `chatgpt-logo-png.png`: ChatGPT Logo
- `claude*.png`: Claude AI相关图片
- `gemini*.png`: Gemini AI相关图片
- `gke*.jpeg`: GKE相关图片（可能用于AI项目）
- `lmstudio.png`: LM Studio相关图片
- `mcp-push.png`: MCP推送相关图片
- `mindmap.png`: 思维导图相关图片
- `opencat.png`: OpenCat相关图片
- `tools.jpeg`: 工具相关图片
- `trae*.png`: TRAE相关图片

### lists/ - 列表文件
- `OpenAI.list*`: OpenAI工具列表
- `claude.list`: Claude AI工具列表
- `o.list`: AI工具列表
- `yahoo.list`: Yahoo相关列表

### html/ - HTML文档
- `verify-glb.html`: GLB验证相关

## 快速检索
- OpenAI工具: 查看 `lists/` 目录中的 `OpenAI.list*` 文件
- AI平台对比: 查看 `lists/` 目录中的各类 `.list` 文件
- Gemini相关: 查看 `images/` 目录中的 `gemini*.png` 图片
- Claude相关: 查看 `images/` 目录中的 `claude*.png` 和 `lists/` 目录中的 `claude.list`
- 概念理解: 查看 `docs/` 目录中的 `concept.md`
- 开发相关: 查看 `docs/` 目录中的 `web-development.md`
- 验证工具: 查看 `docs/` 目录中的 `Verifying-GLB.md` 和 `html/` 目录中的 `verify-glb.html`
- 图片资源: 查看 `images/` 目录
