# Apache 知识库

## 目录描述
本目录包含Apache相关技术栈的知识，包括Apache Airflow工作流管理、Harbor镜像仓库等。

## 目录结构
```
apache/
├── airflow-dag.md           # Airflow DAG相关知识
├── airflow.md               # Apache Airflow相关知识
├── harbor-airflow.md        # Harbor与Airflow集成
└── README.md                # 本说明文件
```

## 文件说明
- `airflow-dag.md`: Apache Airflow中DAG（有向无环图）的创建和管理
- `airflow.md`: Apache Airflow基础概念和使用
- `harbor-airflow.md`: Harbor镜像仓库与Airflow的集成方案

## 快速检索
- Airflow工作流: 查看 `airflow.md` 和 `airflow-dag.md`
- Harbor集成: 查看 `harbor-airflow.md`
