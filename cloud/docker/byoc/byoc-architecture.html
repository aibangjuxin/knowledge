BYOC — Bring Your Own Container

架构与生命周期 / Architecture & Lifecycle

🔵 Source

应用代码 + Dockerfile。决定 build 出什么。放 git，code review 必经。

🟣 CI Build

multi-stage + cache mount。产出 immutable image（tag + digest 双标）。

🟡 Registry

artifact registry / ECR / ACR。lifecycle policy 自动清理 + 漏洞扫描。

🟢 Platform

Vertex AI / Cloud Run / SageMaker / GitHub Actions。拉 image，注入 env，跑。

🔴 Sandbox

containerd 拉的容器。平台合约：$PORT / health endpoint / graceful shutdown。

⚪ Promotion

同一 digest 走 dev → staging → prod。staging 通过才推 prod。
