ListenerSet Multi-Tenant Gateway Architecture

Client → ILB → Shared Gateway → Team Gateway (ListenerSet) → Team Runtime

Shared Gateway Layer

  • • GatewayClass: kgateway (istio)
  • • Gateway CR: shared-gateway
  • • Namespace: kgateway-system
  • • allowedListeners: selector 授权 ListenerSet 绑定
  • • 不直接定义 listeners，所有 listener 由 ListenerSet 提供
  • • 突破 64-listener 限制

Team Gateway Layer (ListenerSet)

  • • ListenerSet CR: team1-gw
  • • Namespace: team1-gateway
  • • hostname: *.team1.appdev.aibang
  • • TLS: mode=Terminate, cert from TLS Secret
  • • TLS Secret: *.team1.appdev.aibang 证书
  • • parentRef: shared-gateway
  • • HTTPRoute parentRef 指向 ListenerSet

Team Runtime Layer

  • • HTTPRoute: 路由规则，parentRef=ListenerSet
  • • Deployment: sidecar=disabled，自有 HTTPS 证书
  • • Pod cert: *.uk.aibang.local (internal PKI)
  • • SVC: ClusterIP, port 8080, targetPort 8443
  • • HPA: min=2, max=10, CPU 70%
  • • NetworkPolicy: default-deny + allow-from-gw
  • • DR: mode=SIMPLE, sni, [caCertificates]

CA / TLS 证书注意事项

  • • Pod 证书 *.uk.aibang.local 由 internal PKI 签发
  • • internal CA 不在 Gateway Envoy 系统 CA 池
  • • 需要将 internal CA 注入 ingressgateway Pod
  • • DestinationRule 加 caCertificates 字段
  • • 或使用 insecureSkipVerify: true（仅临时）
  • • Gateway TLS Secret (*.team1.appdev) 通常用公共 CA

HTTPRoute → ListenerSet 绑定链

  • • HTTPRoute.parentRefs.kind = ListenerSet
  • • HTTPRoute.parentRefs.name = team1-gw
  • • ListenerSet.parentRef = shared-gateway (Gateway)
  • • ListenerSet.listeners 定义 port/TLS/hostname
  • • Gateway.allowedListeners 控制哪些 NS 可绑定
  • • 三层双向握手：HTTPRoute ↔ ListenerSet ↔ Gateway

流量加密完整链路

  • • ① Client → ILB: HTTPS (公共 CA，浏览器校验)
  • • ② ILB → Shared Gateway: HTTPS (team 级别证书)
  • • ③ Shared Gateway → Team ListenerSet: SNI 路由
  • • ④ TLS Termination (ListenerSet 用 team 证书终止)
  • • ⑤ Gateway → Runtime: TLS Origination (DR SIMPLE)
  • • ⑥ Runtime Pod: app 终止 HTTPS 8443 (internal CA)
