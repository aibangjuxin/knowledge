K8s Gateway API — Multi-Tenant E2E Flow

GKE + K8s Gateway (ListenerSet) + Kong DP · Direct Container vs Kong-API Routing

Flow 1 — Direct Container

  • • Client → ILB (GCP) → Envoy (abjx-gw-int)
  • • Envoy → HTTPRoute (teamname-int)
  • • HTTPRoute → teamname-svc (ClusterIP)
  • • → Direct Container Pod (apigateway: NONE)
  • • NetPol: default-deny + allow-ingress-gw
  • • Envoy default timeout: 5s → set DR.timeout

Flow 2 — Kong DP Routing

  • • Client → ILB → Envoy (abjx-gw-int)
  • • Envoy → HTTPRoute → Kong DP (kong-apw-kong-int)
  • • Kong DP → Kong-API Pod (apigateway: KONG)
  • • NetPol: default-deny + allow-ingress-int-kdp
  • • Kong handles: auth, rate-limit, transform
  • • Kong plugin chain per route/service

Control Plane — ListenerSet Binding

  • • Gateway (infrastructure) → allowedListeners selector
  • • ListenerSet (listenersets) → parentRef: shared-gateway
  • • HTTPRoute (teamname-int) → parentRef: ListenerSet
  • • allowedRoutes.namespaces: tenant label matching
  • • ReferenceGrant for cross-ns backendRefs
  • • Security: namespace not in allowedRoutes → REJECT

Security — NetworkPolicy

  • • abjx-gw-int: default-deny + allow-dns/istiod/gcp-hc
  • • teamname-int: default-deny + allow-egress-dns/drn
  • • apigateway:NONE: ingress from gw-ns only
  • • apigateway:KONG: ingress from kong-apw-kong-int
  • • type:ms: ingress from ms/KONG/NONE
  • • NONE pod egress: ONLY other NONE pods (no KONG)

Timeouts & Resilience

  • • HTTPRoute.spec.rules[].timeouts.request (Extended)
  • • HTTPRoute.spec.rules[].timeouts.backendTimeout
  • • DestinationRule.trafficPolicy.timeout (覆盖 Envoy 5s)
  • • DestinationRule.trafficPolicy.connectTimeout (10s)
  • • Envoy UT flag → upstream timeout
  • • Envoy UC flag → connection failure

GCP ILB Health Check

  • • GCP LB HC source: 35.191.0.0/16 + 130.211.0.0/22
  • • Envoy port 15021 (health check endpoint)
  • • NetPol: default-allow-gcp-hc-ingress required
  • • externalTrafficPolicy: Cluster → SNAT applies
  • • GKE Node CIDR: 192.168.64.0/19 must be whitelisted
  • •，否则 Envoy nodes marked unhealthy → 504
