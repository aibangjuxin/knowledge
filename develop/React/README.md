# React

React 开发相关知识库。

## 目录结构

```
React/
├── docs/                  # React 文档
├── k8s/                   # K8s 部署配置
├── public/                # 静态资源
├── scripts/               # 构建脚本
├── src/                   # 源码
├── Dockerfile
├── README.md
└── README_knowledge.md
```

## 子目录内容

- `docs/`: README.md, README_knowledge.md, nginx.conf, package.json
- `k8s/`: deployment.yaml, hpa.yaml, service.yaml
- `public/`: index.html
- `scripts/`: build.sh, deploy.sh, merged-scripts.md
- `src/`: App.css, App.js, index.css, index.js
