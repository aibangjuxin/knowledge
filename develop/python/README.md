# Python 知识库

## 目录描述
本目录包含Python编程语言相关的知识、实践经验、代码示例和工具脚本。

## 目录结构
```
python/
├── docs/                     # Markdown文档
├── examples/                 # 示例文件
├── scripts/                  # Python脚本文件
└── README.md                 # 本说明文件
```

## 子目录说明 (按功能分类)
### scripts/ - Python脚本
- `base-syntax.py`: Python基础语法示例
- `get_domain.py`: 域名获取相关代码
- `time.py`, `time2.py`, `time3.py`, `time4.py`: 时间处理相关代码

### docs/ - 文档文件
- `gunicorn-fastapi.md`: Gunicorn与FastAPI集成
- `python-uvicorn.md`: Uvicorn相关配置
- `python-cronjob.md`: Python定时任务
- `bigquery-*.md`: BigQuery相关文档
- `time*.md`: 时间处理相关文档

### examples/ - 示例文件
- `*.txt`: 文本示例文件

## 快速检索
- Python基础: 查看 `scripts/` 目录中的 `base-syntax.py`
