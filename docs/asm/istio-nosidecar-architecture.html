Istio Ingress Gateway (No Sidecar) 架构图

GKE + ASM Managed Control Plane + TLS Termination at Gateway

关键设计决策

  • • Gateway namespace: istio.io/rev=asm-managed
  • • Runtime namespace: istio-injection: disabled
  • • TLS termination at Gateway (file-mounted certs)
  • • TLS origination to backend (SIMPLE mode)
  • • No mTLS (runtime has no sidecar)

端口映射 (Service)

  • • LB :443 → Gateway Pod :8443
  • • Gateway :8443 (Envoy listener)
  • • Runtime :8443 (nginx HTTPS)
  • • Gateway readiness: :15021/healthz
  • • Envoy admin: :15000

验证命令

  • • curl --resolve 'test.team-a.appdev.aibang:443:34.147.243.168' https://test.team-a.appdev.aibang/
  • • kubectl get gatewayclass (检查 istio)
