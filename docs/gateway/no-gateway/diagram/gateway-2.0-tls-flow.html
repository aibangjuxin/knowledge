❌ 禁止用法 · 原文档错误项
❌ RequestHeaderModifier 修改 Host
Host 是 forbidden header
规范明确禁止通过 RequestHeaderModifier 修改
✅ 改用 URLRewrite.hostname
❌ %{request.host}% 变量模板
GKE Gateway 不支持 Envoy/Nginx 风格变量
支持格式: {client_region}（Google Cloud LB）
✅ 改用静态值 X-Original-Host: api2.appdev.abjx
❌ rules.matches.hostname 无效字段
hostname 不在 rules.matches 里
应放在 spec.hostnames 层
✅ spec.hostnames → rules[].matches[]
❌ RequestHeaderModifier.set 写成 map
错误: set: { Host: xxx }
✅ 正确: set: [{ name: X-Upstream, value: xxx }]
必须使用 name/value 数组格式
