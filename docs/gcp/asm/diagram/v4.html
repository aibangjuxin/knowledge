Istio No-Sidecar — Same Namespace Pattern (v4)

Gateway + HTTPRoute + DestinationRule 同 Namespace | Runtime Namespace 仅含 Service

💡 核心洞察: 所有 Istio 控制面资源 (Gateway, HTTPRoute, DestinationRule) 可以放在同一个 Namespace 中 — 这就是最小工作集。Runtime Namespace 只放 Service 和 Pod。

最小工作集 (3 Resources)

  • • Gateway — TLS 终止 + 监听
  • • HTTPRoute — 路由规则
  • • DestinationRule — TLS re-encrypt
namespace: tenant-a-gateway-ns
  Gateway → tenant-a-gw
  HTTPRoute → app-route
  DestinationRule → tls-app
  Secret → tls-cert (同 NS)

Runtime Namespace

  • • namespace: tenant-a-runtime-ns
  • • Service → app (ClusterIP)
  • • Pod → nginx/httpd
  • • Secret → app-tls (backend cert)
  • • istio-injection: disabled
  • • 无任何 Gateway API 资源

ReferenceGrant (何时需要)

  • • 需要: HTTPRoute → 跨 NS Service
  • • 不需要: Gateway → 同 NS Secret
  • • 不需要: HTTPRoute + Gateway 同 NS
namespace: tenant-a-runtime-ns
ReferenceGrant: 允许
  HTTPRoute (tenant-a-gateway-ns) → Service

多租户扩展

  • • 每租户一个 *-gateway-ns
  • • 每租户一个 *-runtime-ns
  • • Gateway NS 之间完全隔离
  • • 共享 GCP ILB (via GatewayClass)
  • tenant-a: GW + HR + DR → runtime:Service
  • tenant-b: GW + HR + DR → runtime:Service
  • tenant-c: GW + HR + DR → runtime:Service

YAML 布局示例 (同一 Namespace)

# tenant-a-gateway-ns
---
apiVersion: gateway.networking.k8s.io/v1
kind: Gateway
metadata:
  name: tenant-a-gw
  namespace: tenant-a-gateway-ns
spec:
  gatewayClassName: istio
  listeners:
    - name: https
      hostname: "*.tenant-a.domain"
      port: 443
      protocol: HTTPS
      tls:
        mode: Terminate
        certificateRefs:
          - name: tls-cert        # 同 NS，无需 RefGrant
            namespace: tenant-a-gateway-ns

---
apiVersion: gateway.networking.k8s.io/v1
kind: HTTPRoute
metadata:
  name: app-route
  namespace: tenant-a-gateway-ns  # 同 NS，无需 RefGrant
spec:
  parentRefs:
    - name: tenant-a-gw
  hostnames:
    - "app.tenant-a.domain"
  rules:
    - backendRefs:
        - name: app
          namespace: tenant-a-runtime-ns  # 跨 NS，需要 RefGrant
          port: 443

---
apiVersion: networking.istio.io/v1beta1
kind: DestinationRule
metadata:
  name: tls-app
  namespace: tenant-a-gateway-ns  # 必须与 Gateway 同 NS
spec:
  host: app.tenant-a-runtime-ns.svc.cluster.local
  trafficPolicy:
    portLevelSettings:
      - port:
          number: 443
        tls:
          mode: SIMPLE
          insecureSkipVerify: true
