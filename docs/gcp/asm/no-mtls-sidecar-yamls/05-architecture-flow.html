ASM No-Sidecar Gateway Architecture

ASM 作为纯网关 / Runtime Pod 不注入 Sidecar 的流量架构

流量架构总览 / Traffic Flow Overview

流量分段详解 / Traffic Segment Breakdown

Gateway Namespace (ASM Managed)

  • • Namespace label: istio.io/rev: asm-managed
  • • Gateway Pod 包含 Envoy sidecar
  • • Gateway CRD: SIMPLE mode, credentialName
  • • VirtualService: Host/path 路由
  • • DestinationRule: tls.mode: SIMPLE → 8443

Runtime Namespace (No Sidecar)

  • • Namespace label: istio-injection: disabled
  • • Deployment annotation: sidecar.istio.io/inject: "false"
  • • App Container 监听 HTTPS 8443
  • • TLS Secret 挂载: /etc/tls
  • • NetworkPolicy: 默认拒绝, 只允许 Gateway ns

安全边界说明

  • • SPIFFE 工作负载身份: ✗ 牺牲
  • • 自动 mTLS 证书轮转: ✗ 牺牲
  • • L7 AuthorizationPolicy: ✗ 不可用
  • • L3/L4 NetworkPolicy: ✓ 必须
  • • 应用层证书认证: ✓ 必须
这是安全能力降级方案，适合迁移过渡，不建议作为长期平台默认模型。
