跨项目 PSC 架构全景图

A Project Shared VPC (Consumer) → PSC Tunnel → B Project Private VPC (Producer)

PSC NEG (Consumer)

  • • type: PRIVATE_SERVICE_CONNECT
  • • 指定 subnet 分配 PSC 接入 IP
  • • 无需暴露 Backend 真实 IP
  • • 属于 A Project Shared VPC

Service Attachment (Producer)

  • • 绑定 PSC NAT Subnet（必须）
  • • connection-preference: ACCEPT_MANUAL
  • • consumer-accept-list 控制访问权限
  • • 属于 B Project Private VPC

PSC 跨 VPC 隧道

  • • GCP 内部网络自动建立
  • • 无需 VPC Peering
  • • IP 地址空间可重叠
  • • SNAT 实现双向 IP 隔离
