# Gateway 知识库

## 目录描述
本目录包含API网关相关的知识、设计模式和实现方案。

## 目录结构
```
gateway/
├── no-gateway/              # 无网关方案相关内容
├── Design.md                # 网关设计相关文档
└── README.md                # 本说明文件
```

## 文件说明
- `Design.md`: API网关设计原则和模式
- `no-gateway/`: 无网关架构或直接访问方案相关内容

## 快速检索
- 网关设计: 查看 `Design.md`
- 无网关方案: 查看 `no-gateway/` 目录
