Istio 出口流量架构

ServiceEntry 注册外部服务 · Egress Gateway 出口网关 · 出口流量可视化

模式A: 直接出口

  • • Sidecar Envoy 直接连接外部服务
  • • ServiceEntry 注册外部服务到网格
  • • mTLS 自动在 sidecar 间建立
  • • 适合无合规要求的出口流量

模式B: Egress Gateway

  • • 专用出口网关终止 TLS/mTLS
  • • 集中式出口策略和日志
  • • 支持出口流量镜像/注入
  • • 满足金融/合规审计要求

ServiceEntry

  • • 将外部 API/SaaS/DB 注册为网格内服务
  • • location: MESH_EXTERNAL / REGISTRY_ONLY
  • • 配合 DestinationRule 配置 TLS
  • • 配合 AuthorizationPolicy 出口白名单
