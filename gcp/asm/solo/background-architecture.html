ASM Solo Gateway — 3-NS Multi-Tenant Traffic Flow

Gateway API (Istio GW Class) + ListenerSet + HTTPRoute · 两段 TLS(terminate + re-encrypt) · 平台侧冻结,租户侧可复制

L1 Platform — abjx-gw-int

  • Gateway CR (gatewayClassName: istio)
  • 持有 listener 占位,真正的 listener 由 ListenerSet 注入
  • spec.allowedRoutes.from = ListenerSet + namespace selector 锁定租户 NS
  • 改变频率:0(收工后冻结)

L2 ListenerSet — abjx-listenerset-int

  • ListenerSet CR (hostname 通配 + TLS Terminate)
  • 引用当前 NS 内的 wildcard TLS Secret
  • parentRef → abjx-gw-int/Gateway(扩展 listener)
  • 改变频率:0(冻结),但新 hostname 需在原 ListenerSet 上加 rule 或新建一个

L3 Tenant Runtime — 唯一可复制层

  • HTTPRoute(parentRef 指回 ListenerSet,不是 Gateway)
  • Service (ClusterIP) + Deployment + Pods
  • DestinationRule (TLS re-encrypt / outlier / pool)
  • pod 用的通配证书如需,需跨 NS 复制 Secret(json + jq)
