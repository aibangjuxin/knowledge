GKE Pod 调用 PV 存储流程

master GCP Project · Single Tenant per namespace · KMS dynamic injection · BigQuery 归因

① 业务方视角

  • Container 写 /var/lib/mysql
  • kubelet 负责把目录挂到 Pod
  • 看起来"就是个目录",不关心 GCP 实际在哪
  • apply 一个 PVC + Deployment 就结束

② K8s 资源链

  • PVC(命名空间内)→ PV(cluster 级)
  • SC prod-db-ssd 是平台公共 SC
  • PV 通过 csi.volumeHandle 绑 PD selfLink
  • SC 内不 hardcode KMS ★

③ 平台 + GCP 落地

  • Webhook 查 Tenant CRD 拿 KMS + 注入 annotation
  • 自研 Provisioner 把 KMS 写入 compute.disks.insert
  • PD 落在 master project,labels 带 tenant/customer/api-key
  • BigQuery 按 labels 聚合做月度归因
