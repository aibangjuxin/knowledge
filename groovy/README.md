# Groovy 知识库

## 目录描述
本目录包含Groovy编程语言相关的知识、代码示例和Jenkins集成。

## 目录结构
```
groovy/
├── docs/                     # Markdown文档
├── groovyfiles/              # Groovy源代码文件
├── scripts/                  # Shell脚本
└── README.md                 # 本说明文件
```

## 子目录说明 (按功能分类)
### docs/ - 文档
- `jenkins-agent.md`: Jenkins Agent相关
- `read-each-line.md`: 逐行读取相关
- `simple-groovy.md`: 简单Groovy示例
- `p2l.md`: 可能是参数到列表相关文档
- `command.txt`: 命令相关文本文件
- `grepkey.txt`: 关键字搜索相关文本文件
- `cursorgroovy.code-workspace`: VSCode工作区配置文件

### groovyfiles/ - Groovy文件
- `3.groovy`: Groovy代码示例
- `command.groovy`: 命令执行相关Groovy脚本
- `groovy-group-regex.groovy`: Groovy正则表达式分组
- `ios.groovy`: iOS相关Groovy代码
- `lex-test.groovy`: 测试相关Groovy代码
- `mapping.groovy`: 映射相关Groovy代码
- `parameters-blank.groovy`: 参数处理相关
- `simple-groovy.md`: 简单Groovy示例
- `time.groovy`: 时间处理相关
- `utilsFile.groovy`: 工具类文件
- `temp.groovy`: 临时Groovy文件

### scripts/ - 脚本
- `git.sh`: Git相关脚本

## 快速检索
- Groovy基础: 查看 `docs/` 目录中的 `simple-groovy.md`
- 正则表达式: 查看 `groovyfiles/` 目录中的 `groovy-group-regex.groovy`
- Jenkins集成: 查看 `docs/` 目录中的 `jenkins-agent.md` 和 `groovyfiles/` 目录中的 `parameters-blank.groovy`
- 文本处理: 查看 `docs/` 目录中的 `read-each-line.md`
- 时间处理: 查看 `groovyfiles/` 目录中的 `time.groovy`
- 脚本: 查看 `scripts/` 目录
