# ISH 知识库

## 目录描述
本目录包含ISH（可能是iOS Shell或其他相关技术）相关的知识、API列表和Shell更新。

## 目录结构
```
ish/
├── docs/                     # Markdown文档
├── text/                     # 文本文件
└── README.md                 # 本说明文件
```

## 子目录说明 (按功能分类)
### docs/ - 文档
- `a-shell.md`: A-Shell相关文档
- `update-oh-zsh.md`: Oh-My-Zsh更新相关文档

### text/ - 文本文件
- `api-list.txt`: API列表文件

## 快速检索
- Shell相关: 查看 `docs/` 目录中的 `a-shell.md`
- Shell更新: 查看 `docs/` 目录中的 `update-oh-zsh.md`
- API列表: 查看 `text/` 目录中的 `api-list.txt`
