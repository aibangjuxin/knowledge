# Java Code 知识库

## 目录描述
本目录包含Java代码项目、Maven配置和Java基础构建相关的知识。

## 目录结构
```
java-code/
├── .mvn/                     # Maven配置
├── config/                   # 配置文件
├── docs/                     # Markdown文档
├── scripts/                  # 脚本文件
├── src/                      # 源代码
├── target/                   # 编译目标
└── README.md                 # 本说明文件
```

## 子目录说明 (按功能分类)
### docs/ - 文档
- `Java-base-build.md`: Java基础构建相关文档

### scripts/ - 脚本
- `mvnw`: Maven包装器脚本

### config/ - 配置文件
- `pom.xml`: Maven项目对象模型配置文件

### .mvn/ - Maven配置
- Maven相关配置文件

### src/ - 源代码
- Java源代码文件

### target/ - 编译目标
- Maven编译后的目标文件

## 快速检索
- Java构建: 查看 `docs/` 目录中的 `Java-base-build.md`
- Maven配置: 查看 `config/` 目录中的 `pom.xml`
- Maven脚本: 查看 `scripts/` 目录中的 `mvnw`
- 源代码: 查看 `src/` 目录
- Maven配置: 查看 `.mvn/` 目录
