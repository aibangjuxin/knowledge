# Java 知识库

## 目录描述
本目录包含Java编程语言相关的知识、实践经验、框架使用和解决方案。该目录结构已按主题分类为多个子目录。

## 目录结构
```
java/
├── build-tools/              # 构建工具相关 (Maven, Gradle等)
├── debug/                    # 调试相关技巧和工具
├── encoding/                 # 编码相关问题和解决方案
├── general/                  # Java通用知识和概念
├── java-appd/                # Java与AppDynamics集成
├── java-auth/                # Java认证授权相关
├── java-core/                # Java核心概念和特性
├── java-pom/                 # POM文件相关配置
├── java-scan/                # Java代码扫描相关
├── java-spring/              # Java Spring框架相关
├── scripts/                  # Java相关脚本
├── spring/                   # Spring框架专门内容
└── README.md                 # 本说明文件
```

## 子目录说明
### java-core/ - Java核心知识
包含Java语言核心概念、内存管理、编码等基础知识。
- `java-memory.md`: Java内存管理
- `java-encoding.md`: Java编码问题
- `Java-Process-and-Thread.md`: Java进程和线程

### java-spring/ & spring/ - Spring框架
包含Spring框架的使用和最佳实践。

### build-tools/ - 构建工具
包含Maven、Gradle等构建工具的配置和使用。

### debug/ - 调试技巧
包含Java应用调试相关技巧和工具。

## 快速检索
- Java核心概念: 查看 `java-core/` 目录
- Spring框架: 查看 `java-spring/` 和 `spring/` 目录
- 构建配置: 查看 `build-tools/` 和 `java-pom/` 目录
- 调试技巧: 查看 `debug/` 目录
- 认证授权: 查看 `java-auth/` 目录
- 应用监控: 查看 `java-appd/` 目录
