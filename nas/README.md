# NAS 知识库

## 目录描述
本目录包含网络附加存储（NAS）、Docker网络、K3s集群和相关技术的知识。

## 目录结构
```
nas/
├── docs/                     # Markdown文档
└── README.md                 # 本说明文件
```

## 子目录说明 (按功能分类)
### docs/ - 文档
- `Docker-network*.md`: Docker网络相关文档（包括中文版和Claude版本）
- `Docker-network*.html`: Docker网络相关HTML文档（包括中文版）
- `docker-proxy*.md`: Docker代理相关
- `docker-proxy.html`: Docker代理相关HTML文档
- `docker-network-flow.md`: Docker网络流程相关
- `How-to-flow-k3s-logs.md`: K3s日志流程相关
- `local-images-error.md`: 本地镜像错误相关
- `start-qnap-k3s.md`: QNAP K3s启动相关
