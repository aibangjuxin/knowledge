# NBA 知识库

## 目录描述
本目录包含NBA数据分析、Python脚本和调试相关的知识。

## 目录结构
```
nba/
├── data/                     # 数据文件
├── docs/                     # Markdown文档
├── scripts/                  # Python脚本
└── README.md                 # 本说明文件
```

## 子目录说明 (按功能分类)
### docs/ - 文档
- `python-debug.md`: Python调试相关文档

### scripts/ - 脚本
- `compare.py`: 比较脚本
- `nba.py`: NBA数据处理脚本

### data/ - 数据文件
- NBA相关数据文件

## 快速检索
- Python调试: 查看 `docs/` 目录中的 `python-debug.md`
- 数据处理脚本: 查看 `scripts/` 目录中的 `nba.py`
- 比较脚本: 查看 `scripts/` 目录中的 `compare.py`
- 数据文件: 查看 `data/` 目录
