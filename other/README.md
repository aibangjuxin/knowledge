# Other 知识库

## 目录描述
本目录包含各种其他技术主题、工具使用和杂项知识。

## 目录结构
```
other/
├── docs/                     # Markdown文档
├── scripts/                  # Shell脚本
└── README.md                 # 本说明文件
```

- `mermaid.md`: Mermaid图表语法
- `proxy.md`: 代理配置相关

- `java-cert.txt`: Java证书相关

- `git.sh`: Git相关脚本
