# PPT (PowerPoint) 知识库

## 目录描述
本目录包含PowerPoint演示文稿、相关文档和演示内容。

## 目录结构
```
ppt/
├── docs/                     # Markdown文档
├── presentations/            # PowerPoint演示文稿
└── README.md                 # 本说明文件
```

## 子目录说明 (按功能分类)
### docs/ - 文档
- `appd*.md`: AppDynamics相关文档
- `recaptcha.md`: reCAPTCHA相关文档
- `svg.md`: SVG相关文档

### presentations/ - 演示文稿
- `appd*.pptx`: AppDynamics相关演示文稿
- `reCAPTCHA.pptx`: reCAPTCHA相关演示文稿
- `APPD.pptx`: AppDynamics相关演示文稿

## 快速检索
- AppDynamics相关: 查看 `docs/` 目录中的 `appd*.md` 文件和 `presentations/` 目录中的 `appd*.pptx` 演示文稿
- reCAPTCHA相关: 查看 `docs/` 目录中的 `recaptcha.md` 和 `presentations/` 目录中的 `reCAPTCHA.pptx`
- SVG相关: 查看 `docs/` 目录中的 `svg.md`
- 演示文稿: 查看 `presentations/` 目录
