# Report 知识库

## 目录描述
本目录包含报告、数据分析、BigQuery查询、成本计算和监控相关的知识和工具。

## 目录结构
```
report/
├── docs/                     # Markdown文档
├── html/                     # HTML页面和工具
├── scripts/                  # 脚本文件
└── README.md                 # 本说明文件
```

## 子目录说明 (按功能分类)
### docs/ - 文档
- `api-cost*.md`: API成本相关文档
- `bigquery-*.md`: BigQuery相关文档
- `looker-studio*.md`: Looker Studio相关文档
- `jira-*.md`: JIRA相关文档
- `report-*.md`: 报告相关文档

### html/ - HTML页面和工具
- `gke-calculator-*.html`: GKE成本计算器页面
- `calc*.html`: 计算工具页面
- `5time*.html`: 时间相关工具页面

### scripts/ - 脚本
- `api-count.py`: API计数脚本
- `p*.py`: Python脚本

## 快速检索
- BigQuery相关: 查看 `docs/` 目录中的 `bigquery-*.md` 文件
- 成本计算: 查看 `docs/` 目录中的 `api-cost*.md` 文件和 `html/` 目录中的 `gke-calculator-*.html` 文件
- Looker Studio: 查看 `docs/` 目录中的 `looker-studio*.md` 文件
- JIRA集成: 查看 `docs/` 目录中的 `jira-*.md` 文件
- 计算工具: 查看 `html/` 目录
- 脚本工具: 查看 `scripts/` 目录
