Pricing Formula
This calculator estimates platform usage fees based on the following pricing formula:
  • Base Cost: $180/month (platform infrastructure fee)
  • CPU Cost: $19 × vCPU count × replicas ÷ 2
  • Memory Cost: $19 × memory GB × replicas ÷ 2
  • Total Cost: (Base + CPU + Memory) × API count
