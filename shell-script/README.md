# Shell Script 知识库

## 目录描述
本目录包含Shell脚本编程相关的知识、实用脚本和Bash最佳实践。

## 目录结构
```
shell-script/
├── docs/                     # Markdown文档
├── scripts/                  # Shell脚本
└── README.md                 # 本说明文件
```

## 子目录说明 (按功能分类)
### docs/ - 文档
- `cheatsheets-bash.md`: Bash命令速查表
- `count-time.md`: 时间计算相关
- `default-value.md`: 默认值处理
- `docker.md`: Docker相关脚本知识
- `fun.md`: 函数相关知识
- `source.md`: Source相关文档
- `retry.md`: 重试机制相关
- `shell-set.txt`: Shell设置选项

### scripts/ - 脚本
- `batch_replace*.sh`: 批量替换文件内容脚本
- `git-detail-status.sh`: Git详细状态脚本
- `source.sh`: Source相关脚本
- `git.sh`: Git相关脚本

## 快速检索
- Bash速查: 查看 `docs/` 目录中的 `cheatsheets-bash.md`
- 批量处理: 查看 `scripts/` 目录中的 `batch_replace*.sh`
- 脚本函数: 查看 `docs/` 目录中的 `fun.md`
- 默认值处理: 查看 `docs/` 目录中的 `default-value.md`
- 重试机制: 查看 `docs/` 目录中的 `retry.md`
- Git脚本: 查看 `scripts/` 目录中的 `git-detail-status.sh`
- 脚本加载: 查看 `docs/` 目录中的 `source.md` 和 `scripts/` 目录中的 `source.sh`
