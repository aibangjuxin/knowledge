# Shortcut 知识库

## 目录描述
本目录包含快捷方式、批处理脚本和相关配置文件。

## 目录结构
```
shortcut/
├── config/                   # 配置文件
├── docs/                     # Markdown文档
├── scripts/                  # Shell脚本
├── text/                     # 文本文件
└── README.md                 # 本说明文件
```

## 子目录说明 (按功能分类)
### docs/ - 文档
- `merged-scripts.md`: 合并脚本相关文档
- `trans.md`: 翻译相关文档

### scripts/ - 脚本
- `batch_replace*.sh`: 批量替换脚本
- `a.sh`: 示例脚本

### config/ - 配置文件
- `test.yaml`: 测试配置文件

### text/ - 文本文件
- `a.txt`: 示例文本文件
- `replace.txt`: 替换相关文本文件

## 快速检索
- 批量替换: 查看 `scripts/` 目录中的 `batch_replace*.sh` 脚本
- 配置文件: 查看 `config/` 目录中的 `test.yaml`
- 合并脚本: 查看 `docs/` 目录中的 `merged-scripts.md`
- 文本文件: 查看 `text/` 目录
