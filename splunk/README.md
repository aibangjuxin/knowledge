# Splunk 知识库

## 目录描述
本目录包含Splunk日志分析平台相关的知识、配置和最佳实践。

## 目录结构
```
splunk/
├── docs/                     # Markdown文档
├── scripts/                  # Shell脚本
└── README.md                 # 本说明文件
```

## 子目录说明 (按功能分类)
### docs/ - 文档
- `base.md`: Splunk基础相关
- `Splunk-client-agent.md`: Splunk客户端代理相关
- `splunk-log-format.md`: Splunk日志格式相关
- `Splunk-other.md`: 其他Splunk相关文档

### scripts/ - 脚本
- `git.sh`: Git相关脚本
