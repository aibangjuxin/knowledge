# Squid 知识库

## 目录描述
本目录包含Squid代理服务器相关的知识、配置、部署和优化。

## 目录结构
```
squid/
├── docs/                     # Markdown文档
└── README.md                 # 本说明文件
```

- `codebuddy-enhance.md`: CodeBuddy增强相关
- `gemini-enhance-squid.md`: Gemini增强Squid相关

- `squid-docker.md`: Squid Docker部署

- `squid-log.md`: Squid日志相关
- `squid-proxy-nginx.md`: Squid与Nginx代理集成
- `squid-request_body_passthrough.md`: 请求体透传相关
- `squid-start.md`: Squid启动相关
- `squid-upload.md`: Squid上传相关
