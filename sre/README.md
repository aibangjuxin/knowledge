# SRE (Site Reliability Engineering) 知识库

## 目录描述
本目录包含网站可靠性工程相关的知识、调试方法和运维实践。

## 目录结构
```
sre/
├── docs/                     # Markdown文档
└── README.md                 # 本说明文件
```

## 子目录说明 (按功能分类)
### docs/ - 文档
- `flow-debug-gemini.md`: 使用Gemini进行流程调试

- `how-to-debug-flow.md`: 如何调试流程
- `how-to-debug.md`: 调试方法指南
