# Test 知识库

## 目录描述
本目录包含测试相关的知识、测试类型、性能测试和最佳实践。

## 目录结构
```
test/
├── docs/                     # Markdown文档
├── dr-instance/              # DR实例相关内容
└── README.md                 # 本说明文件
```

## 子目录说明 (按功能分类)
### docs/ - 文档
- `api-testing-types.md`: API测试类型相关
- `api-testing.png`: API测试相关图片
- `BDD.md`: 行为驱动开发测试相关
- `compare-stress-performance-testing.md`: 压力和性能测试比较
- `explorer-ai-test.md`: AI测试探索相关
- `find.md`: 查找相关测试内容
- `hey.md`: Hey工具测试相关
- `improve-api-performance.md`: API性能改进相关
- `nginx-l7-l4.md`: Nginx七层四层测试相关
- `regression.md`: 回归测试相关
- `stress.md`: 压力测试相关
- `Why.md`: 测试原因相关
